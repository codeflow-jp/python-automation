# Python 自動化ツール集

Pythonで作った業務自動化ツールのポートフォリオです。

## 使用技術
- Python 3.14
- openpyxl（Excel操作）
- csv（CSVファイル操作）
- os / datetime（標準ライブラリ）

## ツール一覧

### 売上データ自動集計ツール（sales_tool.py）
CSVファイルの売上データを読み込み、ステータスを自動判定してExcelに書き出すツール。

**できること**
- CSVファイルの自動読み込み
- 月収データをもとにステータスを自動判定
- 結果をExcelファイルに自動書き出し

**使い方**
1. `sales.csv` に名前と月収データを入力
2. `sales_tool.py` を実行
3. `集計結果.xlsx` が自動生成される

## 対応可能な業務
- ExcelやCSVを使った데이터集計・整形の自動化
- ファイルの一括処理
- 業務フローの自動化
