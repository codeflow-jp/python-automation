# CSV自動集計ツール（Python）

CSVファイルを読み込み、データを自動判定してExcelに書き出すツールです。

## 特徴

- CSVファイルのパスをコマンドラインで自由に指定できる
- 判定条件・列名・シート名を config.json で外部管理
- コードを一切変更せず、異なるCSVデータにも対応可能

## 使用技術

- Python 3.14
- openpyxl（Excel出力）
- csv / json / os / sys（標準ライブラリ）

## ファイル構成

- sales_tool.py　← メインスクリプト
- config.json　　← 判定条件・列名の設定ファイル
- sales.csv　　　← 入力データ（サンプル）

## 使い方

### 1. config.json で設定を確認・変更する
```json
{
    "name_column": "名前",
    "value_column": "月収",
    "thresholds": [
        {"label": "目標達成！", "min": 500000},
        {"label": "順調です",   "min": 300000},
        {"label": "軌道中",     "min": 200000},
        {"label": "まだこれから", "min": 0}
    ],
    "sheet_name": "集計結果"
}
```

### 2. スクリプトを実行する
```bash
# デフォルト（sales.csv → 集計結果.xlsx）
python sales_tool.py

# ファイルを指定して実行
python sales_tool.py 任意のデータ.csv 任意の出力ファイル.xlsx
```

### 3. 出力結果（Excel）

| 山田 | 520000 | 目標達成！ |
| 鈴木 | 310000 | 順調です |
| 佐々木 | 210000 | 軌道中 |
| 渡辺 | 80000 | まだこれから |
